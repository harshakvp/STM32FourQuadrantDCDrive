### ATmega328P & STM32-Based Four-Quadrant DC Motor Drive

A hardware implementation of a four-quadrant DC motor drive demonstrating **forward/reverse motoring**, **dynamic braking**, and **PWM-based speed control** using an IR2110-driven MOSFET H-bridge. This repository documents the evolution of the project from an **ATmega328P (Arduino Uno)** implementation to an improved **STM32F446RE** version featuring a modular firmware architecture and a custom-designed PCB.

# Development Journey

The project evolved through two embedded platforms:

### Version 1 — ATmega328P (Arduino Uno)

The initial implementation was developed as part of an Electric Drives course project to understand the fundamentals of H-bridge motor control, PWM generation, and dynamic braking.

### Version 2 — STM32F446RE

The hardware and firmware were redesigned around the STM32F446RE to improve firmware architecture, implement a state-machine-based control strategy, and develop a more reliable and modular hardware platform.

# System Architecture

<p align="center">
  <img src="https://github.com/harshakvp/STM32FourQuadrantDCDrive/blob/3d53636b19a3741cf25f7b09a3cfddf7bbd12060/Documents/Figures/System%20Architecture%20Realization.png">
</p>

The system consists of three functional blocks:

- **Control Stage**
  - ATmega328P / STM32F446RE
  - PWM generation
  - User input processing
  - State-machine control

- **Driver Stage**
  - IR2110 High-Side / Low-Side Gate Drivers

- **Power Stage**
  - IRFZ44N MOSFET H-Bridge
  - Dynamic Braking Circuit
  - 12 V Permanent Magnet DC Motor

# Circuit Schematic

The power stage consists of an IR2110-driven MOSFET H-bridge with a dedicated braking circuit. PWM signals generated by the controller determine the operating quadrant, while the braking circuit safely dissipates regenerative energy through a braking resistor.

---

# Working Principle

1. User inputs determine the operating mode.
2. The controller executes the corresponding state-machine logic.
3. PWM and logic signals are generated for the IR2110 gate drivers.
4. The H-bridge applies the required voltage polarity across the motor.
5. Dynamic braking dissipates regenerative energy through the braking resistor.
6. Dead-time and controlled state transitions prevent shoot-through.
